# 📊 Exploratory Data Analysis (EDA)

## 📖 Overview

This repository contains Jupyter notebooks that perform **Exploratory Data Analysis (EDA)** on different datasets using Python. The notebooks demonstrate data cleaning, preprocessing, visualization, and statistical analysis to uncover meaningful insights.

---

## 📂 Repository Structure

```text
EDA/
├── E2.ipynb
├── EDA1.ipynb
├── EDA_heart.ipynb
├── Untitled.ipynb
├── train.csv
├── test.csv
└── .ipynb_checkpoints/
```

---

## ✨ Features

- Data loading and preprocessing
- Missing value analysis
- Descriptive statistics
- Correlation analysis
- Data visualization
- Outlier detection
- Feature relationship analysis

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/aditiii2/EDA.git
```

### Navigate to the project

```bash
cd EDA
```

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn notebook
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any notebook and run all cells.
